What is this project?

This is a simple AI chatbot built using a Hugging Face language model.
The aim of this project is to understand how chatbots work using Large Language Models (LLMs) and how to connect a backend API with a frontend.

What problem does it solve?

Instead of just showing a UI, this project shows how:

A real AI model is used

A backend API handles user messages

The chatbot gives meaningful replies

How it works (simple flow)

User types a message in the browser

Message goes to FastAPI backend

Backend sends it to Hugging Face LLM

Model reply is shown to the user

Model used

LLaMA-3-8B Instruct

Used through Hugging Face (serverless API)

Chosen because it works without paid endpoints

Tech used

Python

FastAPI

Hugging Face API

HTML, CSS, JavaScript
